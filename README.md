# happy-paws-petshop

Fictional pet shop website developed as part of IME Junior BECOMP training program (Week 3 – Computer Battle track).

## Structure

```
project/
├── index.html              # Home (with slider)
├── pages/
│   ├── login.html          # Login + Sign up (localStorage)
│   ├── services.html       # Services
│   └── shop.html           # Shop (with search)
├── css/
│   └── styles.css          # Complete design system
├── js/
│   ├── main.js             # Mobile menu
│   ├── auth.js             # Login/Sign up via localStorage
│   ├── slider.js           # Automatic carousel
│   ├── products.js         # Product catalog
│   ├── search.js           # Search by name/category/tags
└── assets/images/          # Website images
```

## Features
- ✅ Pixel-perfect layout based on Figma (Home, Services, Shop)
- ✅ Login and registration with validation (localStorage)
- ✅ Functional slider (auto-play + controls + dots)
- ✅ Dynamic search by name, category, or tags
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ No external dependencies except Google Fonts
